# Project Description — Precedent Study: Exposing.AI

## Overview

This website is a **precedent study and critical analysis** of [Exposing.AI](https://exposing.ai/), an investigative research project and counter-surveillance tool created by **Adam Harvey** in collaboration with Jules LaPlace (2021). The site presents written analysis originally authored for a Columbia GSAPP Computational Design Practices course project archive.

The website itself is a separate artifact: a designed container for that analysis, styled to reflect the hacktivist and dystopian surveillance aesthetic of the Exposing.AI project and its promotional poster.

---

## Subject of Analysis: Exposing.AI

**Exposing.AI** exposes the information supply chain behind the global biometrics industry. Building on Harvey's earlier **MegaPixels** initiative (2017), the project reveals how millions of personal photos uploaded to public platforms like Flickr — often under permissive Creative Commons licenses — have been systematically scraped without explicit consent to train, test, and advance AI Facial Recognition Technology (FRT) for academic, commercial, and defense applications.

The live tool allows users to search Flickr metadata to determine whether their images appear in known surveillance datasets, making the invisible infrastructure of biometric data extraction visible and personal.

---

## Analytical Framework

The written analysis in `index.html` is organized into three layers:

1. **Layer 1 — Ontological Analysis**  
   Examines what Exposing.AI *is*: an interactive web-based research tool with dataset documentation, a Flickr search engine, and an about section. Considers Harvey's transparency about matching methodology (text-based, not facial recognition).

2. **Layer 2 — Historical and Contextual Analysis**  
   Situates the project in timeline and culture: the MegaPixels predecessor, the 2020 Clearview AI scandal, Harvey's 2020 Berlin research fellowship, and collaboration with the Surveillance Technology Oversight Project.

3. **Layer 3 — Visual and Aesthetic Analysis**  
   Interprets the site's deliberate use of corporate-tech visual language as critique — positioning Exposing.AI within counter-surveillance art and investigative practices like Forensic Architecture.

---

## Design Intent for This Website

While Exposing.AI adopts a clean, corporate-dashboard aesthetic to subvert big tech's visual identity, **this precedent study site** adopts the dystopian poster imagery associated with the project:

- Giant observers tracking anonymous subjects
- Red detection lines and bounding boxes
- Industrial concrete grays and halftone textures
- Monospace terminal typography

The goal is to create a **hacker-futuristic, dystopian atmosphere** that complements the written analysis — making the reader feel the surveillance environment the project critiques, rather than replicating Exposing.AI's own minimalist tool interface.
